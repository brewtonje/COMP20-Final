Drink Selection

 
Drinks - how to obtain data:
edit the url in index.js example
once you parse objData, you have 100 "strDrink" objects, select one (my example used a random seed)
each "strDrink" object has a
    "strDrink" - Name of drink
    "strDrinkThumb" - url of drink picture
    "idDrink" - id of drink (useless to us)
